# DNA Analyzer

This is a basic bioinformatics project built using Python.

## Features
- Count A, T, G, C bases
- Calculate percentage of each base
- Show output in professional format (e.g., A: 2 (40%))

## Example Output
Input:
AGTCA

Output:
A: 2 (40.0%)
G: 1 (20.0%)
T: 1 (20.0%)
C: 1 (20.0%)

## How to Run
1. Run the Python file
2. Enter DNA sequence
3. Get results instantly
